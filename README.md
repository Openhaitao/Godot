# AI 任务推送器 🚀

一个优雅的浏览器扩展，让你在浏览网页时快速将文本和提示词推送到 Gemini 或 ChatGPT，实现异步处理，不打断工作流。

## ✨ 核心特性

- 🎯 **快捷键触发** - 按 `Ctrl+Shift+G` 快速打开输入框
- 📝 **智能文本捕获** - 自动获取当前选中的文本
- 💬 **自定义提示词** - 输入你想让 AI 做的事情
- 🔄 **自动创建新对话** - 每次推送都在 AI 页面中创建新对话，保持对话整洁
- ⚡ **异步后台处理** - 不打断当前工作，AI 在后台标签页运行
- 📚 **保留历史记录** - 对话直接保存在官方网页的历史记录中

## 🎯 使用场景

- **阅读文章** → 选中段落 → "总结这段文字"
- **遇到英文** → 选中文本 → "翻译成中文"
- **代码审查** → 选中代码 → "解释这段代码的作用"
- **写作优化** → 选中草稿 → "使其更专业"

## 📦 安装步骤

1. 下载或克隆项目到本地
2. 打开浏览器扩展管理页面
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
   - Brave: `brave://extensions/`
3. 开启 **开发者模式**
4. 点击 **加载已解压的扩展程序**
5. 选择项目文件夹
6. 完成！

## 🎨 使用方法

1. 在任意网页选中文本（可选）
2. 按下快捷键 `Ctrl+Shift+G`
3. 输入提示词
4. 选择发送到 Gemini 或 ChatGPT
5. 继续工作，稍后查看结果

## 🔧 技术说明

### 工作原理

1. 监听 `Ctrl+Shift+G` 快捷键
2. 显示优雅的输入框浮层
3. 查找或创建 AI 页面标签页
4. 自动点击"新建对话"按钮
5. 注入消息到输入框并自动发送
6. 在后台运行，不打断当前工作

### 权限说明

- `activeTab` - 读取选中文本
- `scripting` - 向 AI 页面注入脚本
- `tabs` - 管理标签页
- `storage` - 保存用户设置
- `host_permissions` - 访问 Gemini 和 ChatGPT

### 支持的平台

- ✅ Gemini (gemini.google.com)
- ✅ ChatGPT (chatgpt.com)

## ⚙️ 自定义快捷键

1. 进入浏览器扩展管理页面
2. 点击左侧 **键盘快捷键**
3. 找到 **AI 任务推送器**
4. 自定义快捷键
5. 保存

## ⚠️ 注意事项

- 需要先登录 Gemini 和 ChatGPT
- 确保能正常访问这些网站
- 首次使用可能需要手动验证一次

## 🐛 故障排除

**插件无反应？**
- 检查是否已启用插件
- 刷新当前网页
- 查看浏览器控制台

**发送失败？**
- 确保已登录 AI 平台
- 检查网络连接
- 手动打开一次目标网站测试
